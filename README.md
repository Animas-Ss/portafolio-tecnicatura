# Portafolio Profesional - Tecnicatura en Desarrollo de Software

## Descripción del Proyecto
Este proyecto es un portafolio interactivo desarrollado como parte de la **Práctica Formativa Obligatoria 1**. Se trata de una Single Page Application (SPA) que muestra información personal, servicios técnicos, habilidades y proyectos destacados. El objetivo principal es demostrar el dominio de HTML5 semántico, CSS3 avanzado (Flexbox, Grid), tipografía de Google Fonts y principios de diseño responsivo.
